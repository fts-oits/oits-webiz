# OITS Dhaka - Corporate Website

A modern, high-performance single-page application (SPA) for OITS Dhaka, built with React, TypeScript, and Tailwind CSS.

## Tech Stack

*   **Framework**: React 18 + Vite
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS (with custom animations and theme)
*   **Routing**: React Router DOM
*   **Icons**: Lucide React

## Getting Started

### Prerequisites

*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist` folder.

## Project Structure

*   `components/`: Reusable UI components (Hero, Services, Contact, etc.)
*   `ui/`: Generic UI elements (Buttons, Tooltips)
*   `constants.ts`: Static data for services, projects, and testimonials.
*   `types.ts`: TypeScript interfaces.
*   `App.tsx`: Main application layout and routing.

## Features

*   **Responsive Design**: Fully responsive across mobile, tablet, and desktop.
*   **Animations**: Custom Tailwind animations for smooth transitions.
*   **Interactive UI**: Modals, tabs, and carousels.
*   **SEO Friendly**: Clear page structure and semantic HTML.
